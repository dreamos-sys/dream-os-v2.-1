# 🧹 Janitor Indoor Module v3.0

## Features

- ✅ 46 Checklist Items (Toilet: 11, Ruangan: 35)
- ✅ 11 Petugas Assignment- ✅ 3x Daily Check (Pagi/Siang/Sore)
- ✅ AI Smart Schedule (Monthly)
- ✅ Auto-route to Maintenance (if damage)
- ✅ Real-time Sync with Command Center
- ✅ Photo Upload (Before/After)
- ✅ Draft Auto-save

- Tempat Wudhu, Lantai, Floor Drain, Kloset, Plafon, Tempat Sampah

### Ruangan & Area Umum (35 items)

- Sentra Musik/Kreasi, UKS, Psikolog, Lab Kom/IPA, Perpus
- Kepsek, Guru (L/P), Pemasaran, Admin TU, Rapat, CEO, Kabid
- Osis, Mushalla, Inklusi, Gudang Olahraga, Serbaguna, Masjid
- Kantin, Saung (Besar/Kecil), Pos, Gudang Umum, Tangga, Lift
